# Agriculture Decision Intelligence System (ADIS)

## Goal

Help understand agricultural market data and support future decision-making regarding:

- What to grow
- When to sell
- Where to sell

## MVP

Track crop prices across markets and display trends.

## Tech Stack

### Frontend
- React
- TypeScript

### Backend
- FastAPI
- SQLAlchemy
- Alembic

### Database
- PostgreSQL

## Modules

- Crop Module
- Market Module
- Price Module
- Dashboard Module
